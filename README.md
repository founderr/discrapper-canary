# `discrapper-canary`

Welcome to the `discrapper-canary`. the repository which is scraping Discord code updates (canary release - alpha) and efficiently splitting webpack into code chunks. If you're new to Discord Canary, check out the details [here](https://support.discord.com/hc/en-us/articles/360035675191-Discord-Testing-Clients).

## 🚀 Getting Started

### Overview
This repository serves as an archive for Discord updates and facilitates code difference analysis between releases. It was designed to simplify Discord Datamining, providing an easy way to view the code of the latest updates.

### Setup

#### Quick Browsing
If you're only interested in browsing the code, simply press `.` on your keyboard while on the repository page.

#### Advanced Features
For more advanced features such as commit diffing, content searching, code analysis, etc., follow these steps:

1. In the [Codespaces](https://github.com/codespaces) tab on GitHub, create your own codespace. This allows you to connect remotely to this repository using tools like Visual Studio Code.

2. Assuming you are using Visual Studio Code, download the `GitHub Codespaces` and `GitLens` plugins.

3. Go to the Codespaces tab and add `discrapper-canary` as a codespace.

Now you can connect to the `discrapper-canary` codespace and leverage all the functionalities this repository offers!
